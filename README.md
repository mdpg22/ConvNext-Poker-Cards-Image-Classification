## üèó Model Architecture

ConvNeXt is a modernized convolutional neural network architecture that
incorporates design elements inspired by Vision Transformers while
retaining convolutional efficiency.

Key architectural characteristics:

-   Depthwise convolutions
-   Layer normalization
-   Inverted bottleneck blocks
-   Residual connections
-   Scalable design

This model aims to achieve competitive performance with improved
training stability and efficiency.
